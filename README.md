# Computer-networks-project

A LAN (Local Area Network) is a type of network used to connect a group of devices within a 
limited geographical area such as a room, house, or office. the main purpose of this network is to 
enable devices to share data and resources such as printers, files, and internet access efficiently 
and quickly. 
In this setup we’ve created a LAN consisting of a router and three computers the router is the 
central device responsible for assigning IP addresses to each device managing data traffic within 
the network and potentially providing internet access if connected to an external modem.
